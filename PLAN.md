# 리딩수학과학 리디자인 — 작업 계획

## 제품은 두 층으로 나뉜다
- **A층 — 네비게이션/홈** (과목선택, 수학·과학 메인홈, 기본/자유모드, 커리큘럼 레이어, 훈련 메인)
  → 피그마에 아직 **디자인 안 됨**. 컨셉 이미지 + 온보딩 문서로만 존재.
- **B층 — 실제 학습/풀이** (학습화면, 스텝/훈련 결과, 과제센터, 시험대비, 진단평가)
  → 피그마(node 81:2)에 **작업 중 + 컴포넌트 라이브러리 시작됨**.

## 기준
- 기준 화면: **1280 × 720 (태블릿 가로 16:9)**
- 기본 테마: 다크(우주), 라이트 전환 지원
- 과목 3색: 개념 `#0A3D8E` / 유형 `#005752` / 서술형 `#16005D`
- 주 CTA: 오렌지 `#FF7F00` · 정답 `#00CCAE` · 오답 `#FF585B`
- 스타일 금기(디자이너 메모): 그라데이션, 얼룩덜룩·과밀, 복잡한 오브젝트, 16:9 비율 위반

## 정해야 할 것 (진행하며 확인)
1. 화면 복제본이 2~3개씩 있음 → 화면당 완성도 높은 1개를 골라 진행.
2. 진단평가: 온보딩 문서엔 없음 / 피그마엔 있음 → 일단 보류.
3. 펜 필기 툴바: 재작업 중 → 초기엔 제외.
